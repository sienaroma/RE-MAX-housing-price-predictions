# Home Sale Price Predictions for RE/MAX for Three Separate Neighborhoods

### Model Comparison, Cross Validation, Feature Creation and Elimination, Parameter Tuning
#### Models Selected: Random Forest, Ridge, Lasso, and Elastic Net

The final model choice was Lasso as it had the lowest RMSE (0.1146) and highest R^2 (0.8106) and adjusted R^2 (0.7956)
